  <action>User visually verifies all Phase 33 Foundation changes in the browser.</action>
  <what-built>
  Complete Terminal/Hacker visual foundation (Phase 33):
  - Deep dark background (#0d1117) across the entire UI
  - Green accent palette (#3fb950 readable green, #238636 muted green)
  - Fira Code monospace font for the SeedSync logo
  - IBM Plex Sans for all UI text (labels, buttons, navigation)
  - CRT scan-line overlay (subtle horizontal lines across viewport)
  - Custom dark scrollbars (thin, gray thumb, green on hover)
  - Dropdown menus with dark surface background and green hover
  - Form inputs with dark background and green focus states
  </what-built>
  <how-to-verify>
  Open http://localhost:4200 and check these items:

  1. **Overall palette**: Background should be very dark (#0d1117), not Bootstrap's default dark gray (#212529). Text should be light gray (#e6edf3).

  2. **Fonts**: The SeedSync logo should render in Fira Code (monospace). Body text, buttons, and navigation labels should render in IBM Plex Sans (sans-serif). Open browser DevTools > Computed and verify font-family on a button element.

  3. **CRT overlay**: Look closely at the screen — you should see very faint horizontal lines across the entire viewport. They should NOT obstruct readability. If invisible, opacity may need increasing. If too prominent, it may need decreasing.

  4. **Scrollbars**: If the file list is long enough to scroll, scrollbar should be thin with dark track and gray thumb. Hover the thumb — it should turn green.

  5. **Dropdowns**: Click a dropdown (e.g., status filter). The dropdown menu should have a dark surface background (#161b22), and items should highlight green on hover.

  6. **Forms**: Navigate to Settings page. Form inputs should have very dark backgrounds (#0d1117) with subtle borders (#30363d). Click into an input — focus border should be green.

  7. **Selection**: If files exist, select a file row. Selection highlight should be green-tinted, not teal.

  8. **No light mode artifacts**: No white flashes, no light-colored backgrounds, no FOUC on hard refresh.
  </how-to-verify>
