In `file.component.ts`:

2. Add a `@HostBinding('class')` getter that returns a status-based class string for the host element. This enables both the left border (DASH-02) and the glow animation (DASH-04) to target `:host` by status:
   ```typescript
   @HostBinding('class') get hostClass(): string {
       if (!this.file) return '';
       const classes = [`status-${this.file.status}`];
       if (this.file.status === ViewFile.Status.DOWNLOADING) {
           classes.push('downloading-active');
       }
       return classes.join(' ');
   }
   ```

   **IMPORTANT:** The existing `:host.even-row` class is applied by the parent `file-list.component.ts` via `[class.even-row]` binding on the `<app-file>` element. Angular merges `@HostBinding('class')` with attribute-set classes, so `even-row` will continue to work. The `@HostBinding` does NOT override `[class.even-row]` set from the parent.

3. Add a `statusDotClass` getter for DASH-05 (status dot):
   ```typescript
   get statusDotClass(): string {
       return `status-dot dot-${this.file?.status ?? 'default'}`;
   }
   ```

4. Both getters need explicit return types (`:string`) per project lint conventions.
