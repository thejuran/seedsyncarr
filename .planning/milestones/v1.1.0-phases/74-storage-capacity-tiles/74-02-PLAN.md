```python
    @staticmethod
    def _parse_df_output(out: bytes) -> Tuple[Optional[int], Optional[int]]:
        """
        Parse `df -B1 <path>` output. Expected shape:
            Filesystem     1B-blocks       Used Available Use% Mounted on
            /dev/sda1  2000000000000 1300000000000  700000000000 65% /mnt/path

        Returns (total_bytes, used_bytes), or (None, None) on any parse failure
        (silent fallback per D-16 — caller logs at WARN level).
        """
        try:
            text = out.decode("utf-8") if isinstance(out, bytes) else out
        except UnicodeDecodeError:
            return (None, None)
        lines = [line for line in text.splitlines() if line.strip()]
        if len(lines) < 2:
            return (None, None)
        # Last non-empty line is the data row (header is first)
        parts = lines[-1].split()
        if len(parts) < 3:
            return (None, None)
        try:
            total = int(parts[1])
            used = int(parts[2])
        except (ValueError, IndexError):
            return (None, None)
        return (total, used)
```

The parser never raises — all failure modes return `(None, None)`. No logging inside the parser; the caller in `scan()` logs at WARN.

**Step 1 — Add `_should_update_capacity` helper to `Controller` in `src/python/controller/controller.py`.**

Place this private method immediately above `_update_controller_status` (currently at line 620). The helper is pure — no `self` state access:

```python
    @staticmethod
    def _should_update_capacity(old: Optional[int], new: Optional[int]) -> bool:
        """
        >1% change gate (D-12/D-13). Returns True when the new value should be
        written to Status.storage:
          - new is None: never write (caller handles None-to-None as a no-op)
          - old is None and new is not None: always write (first-seen or recovery)
          - otherwise: write only when |new - old| / max(old, 1) > 0.01
        """
        if new is None:
            return False
        if old is None:
            return True
        if old == 0:
            return new != 0
        return abs(new - old) / abs(old) > 0.01
```

**Step 2 — Extend `_update_controller_status` to assign capacity per-side with the gate.**

```python
    def _update_controller_status(self,
                                  remote_scan: Optional[ScannerResult],
                                  local_scan: Optional[ScannerResult]) -> None:
        """
        Update the controller status with latest scan information.

        Writes storage capacity (Phase 74) gated by the >1% change rule (D-12/D-13)
        per-side independently (D-15). A None total_bytes or None used_bytes from a
        scan result leaves the corresponding side untouched (silent fallback per D-16).
        """
        if remote_scan is not None:
            self.__context.status.controller.latest_remote_scan_time = remote_scan.timestamp
            self.__context.status.controller.latest_remote_scan_failed = remote_scan.failed
            self.__context.status.controller.latest_remote_scan_error = remote_scan.error_message
            # Capacity (D-12/D-15): treat (total, used) as a paired measurement.
            if remote_scan.total_bytes is not None and remote_scan.used_bytes is not None:
                old_total = self.__context.status.storage.remote_total
                old_used = self.__context.status.storage.remote_used
                if (Controller._should_update_capacity(old_total, remote_scan.total_bytes)
                        or Controller._should_update_capacity(old_used, remote_scan.used_bytes)):
                    self.__context.status.storage.remote_total = remote_scan.total_bytes
                    self.__context.status.storage.remote_used = remote_scan.used_bytes
        if local_scan is not None:
            self.__context.status.controller.latest_local_scan_time = local_scan.timestamp
            # Capacity (D-13/D-15): independent of remote; paired measurement.
            if local_scan.total_bytes is not None and local_scan.used_bytes is not None:
                old_total = self.__context.status.storage.local_total
                old_used = self.__context.status.storage.local_used
                if (Controller._should_update_capacity(old_total, local_scan.total_bytes)
                        or Controller._should_update_capacity(old_used, local_scan.used_bytes)):
                    self.__context.status.storage.local_total = local_scan.total_bytes
                    self.__context.status.storage.local_used = local_scan.used_bytes
```

**Per-side independence (D-15) is structural here:** the remote block reads only `remote_*` from both the scan result and the status; the local block reads only `local_*`. A `None` on one side has no effect on the other.

**Paired-measurement semantic:** total and used are captured at the same moment by `df -B1` / `shutil.disk_usage`. If either exceeds the 1% gate, BOTH are written atomically so the percentage displayed on the frontend remains self-consistent (a tile showing `old_total` with `new_used` would compute a bogus percentage).

Do NOT add `error_msg` plumbing — D-16 explicitly rejects it. Silent fallback ends at the scanner's WARN log.
