**1. Fix __downloaded_files type (CODE-07):**

In src/python/controller/model_builder.py:

a) The `__init__` initializes `self.__downloaded_files = set()` but the actual runtime value passed via `set_downloaded_files()` is a `BoundedOrderedSet[str]` (from controller_persist.py). The `set()` initial value doesn't support `.touch()` which is called in `_check_deleted_state`.

```python
from common.bounded_ordered_set import BoundedOrderedSet

    ...

    ...
```

```python
def _check_deleted_state(self, model_file: ModelFile) -> None:
    if model_file.state != ModelFile.State.DEFAULT:
        return
    if model_file.local_size is not None:
        return
```

d) Update `clear()` to set `self.__downloaded_files = None` instead of `self.__downloaded_files.clear()` (since it may be a shared reference to the persist object's BoundedOrderedSet, clearing it would wipe persisted state).

The current `self.__downloaded_files.clear()` would actually mutate the persist object's BoundedOrderedSet. This is likely a bug. Change clear() to reset the reference:

```python
    ...
```

e) Remove `Set` from the typing imports if no longer needed (check other usages first).

**2. Fix directory DOWNLOADED edge case (CODE-12):**

In `_are_all_children_downloaded`, the current logic returns `True` for a directory that has NO remote children (the while loop never enters the `if not child_file.is_dir and child_file.remote_size is not None:` branch). This means an empty directory or a directory with only subdirectories (no leaf files) would be marked DOWNLOADED even though nothing was actually downloaded.

Fix: Track whether any downloadable child was found. If no downloadable children exist, the directory should stay DEFAULT:

```python
def _are_all_children_downloaded(self, model_file: ModelFile) -> bool:
    frontier = list(model_file.get_children())
    has_downloadable_children = False

    while frontier:
        child_file = frontier.pop(0)
        if not child_file.is_dir and child_file.remote_size is not None:
            has_downloadable_children = True
            if child_file.state != ModelFile.State.DOWNLOADED:
                return False
        frontier.extend(child_file.get_children())

    return has_downloadable_children
```

This ensures a directory with no downloadable leaf files stays in DEFAULT state.

```python
def _set_import_status(self, model: Model, file_name: str) -> None:
    """
    Set import_status to IMPORTED on a model file if not already set.
    Creates a mutable copy, updates status, and writes back to model.

    Args:
        model: The model to update (may be the live model or a new model being built)
        file_name: Name of the file to update

    try:
        file = model.get_file(file_name)
        if file.import_status != ModelFile.ImportStatus.IMPORTED:
            new_file = copy.copy(file)

            new_file.import_status = ModelFile.ImportStatus.IMPORTED
            model.update_file(new_file)
    except ModelError:
        pass
```
