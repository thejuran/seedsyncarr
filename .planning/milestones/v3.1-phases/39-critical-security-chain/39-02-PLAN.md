<objective>
Replace pickle serialization with JSON in the remote scanner pipeline, eliminating the CWE-502 deserialization RCE vector.

Purpose: Pickle deserialization of untrusted data allows arbitrary code execution. The scan_fs script runs on the remote server and sends results over SSH. If the remote server is compromised, an attacker could craft pickle payloads that execute arbitrary code on the local SeedSync host. JSON is a safe, data-only format that cannot trigger code execution.

Output: scan_fs.py outputs JSON, remote_scanner.py consumes JSON, SystemFile has JSON serialization methods, all tests updated.
</objective>

   a. `to_dict(self) -> dict` — instance method that returns a JSON-serializable dict:
      ```python
      def to_dict(self) -> dict:
          """Convert to JSON-serializable dictionary."""
          d = {
              "name": self.__name,
              "size": self.__size,
              "is_dir": self.__is_dir,
          }
          if self.__timestamp_created is not None:
              d["time_created"] = self.__timestamp_created.isoformat()
          if self.__timestamp_modified is not None:
              d["time_modified"] = self.__timestamp_modified.isoformat()
          if self.__children:
              d["children"] = [child.to_dict() for child in self.__children]
          return d
      ```

   b. `from_dict(cls, data: dict) -> "SystemFile"` — classmethod that reconstructs a SystemFile:
      ```python
      @classmethod
      def from_dict(cls, data: dict) -> "SystemFile":
          """Reconstruct SystemFile from a dictionary."""
          from datetime import datetime
          time_created = None
          time_modified = None
          if "time_created" in data and data["time_created"] is not None:
              time_created = datetime.fromisoformat(data["time_created"])
          if "time_modified" in data and data["time_modified"] is not None:
              time_modified = datetime.fromisoformat(data["time_modified"])
          sf = cls(
              name=data["name"],
              size=data["size"],
              is_dir=data.get("is_dir", False),
              time_created=time_created,
              time_modified=time_modified,
          )
          for child_data in data.get("children", []):
              sf.add_child(cls.from_dict(child_data))
          return sf
      ```

   c. Add `import json` at the top of the file (needed for downstream usage, but the methods themselves only need `dict`). Actually, `json` is not needed here — just the `datetime` import which already exists. No new imports needed.
