In `web_app.py`, add a Bottle `after_request` hook (using `@self.hook('after_request')`) in the `__init__` method to set security headers on every response. Bottle's `after_request` hook runs after every request handler.

Add this in `WebApp.__init__` after the existing initialization:

```python
@self.hook('after_request')
def _add_security_headers():
    bottle.response.set_header('Content-Security-Policy', "default-src 'self'; script-src 'self'; style-src 'self' 'unsafe-inline' https://fonts.googleapis.com; font-src 'self' https://fonts.gstatic.com; connect-src 'self'; img-src 'self' data:; frame-ancestors 'none'")
    bottle.response.set_header('X-Frame-Options', 'DENY')
    bottle.response.set_header('X-Content-Type-Options', 'nosniff')
```

The CSP policy:
- `default-src 'self'` — only load resources from same origin
- `script-src 'self'` — scripts only from same origin
- `style-src 'self' 'unsafe-inline' https://fonts.googleapis.com` — styles from self + inline (Bootstrap needs this) + Google Fonts
- `font-src 'self' https://fonts.gstatic.com` — fonts from self + Google Fonts CDN
- `connect-src 'self'` — AJAX/SSE only to same origin
- `img-src 'self' data:` — images from self + data URIs
- `frame-ancestors 'none'` — cannot be embedded in frames (complements X-Frame-Options)
