### Integration Gap (1)

**CONF-04: Settings UI → Config API redaction**
- `serialize_config.py` redacts lftp topology fields → **REDACTED**
- `ConfigService.getConfig()` populates BehaviorSubject FROM the redacted API response
- `settings-page.component.html` binds text inputs to BehaviorSubject values
- Result: Server Address, Server User, Server Directory display "**REDACTED**"
- Risk: User saving without clearing writes "**REDACTED**" as actual config value

### Cross-Phase Dependencies (Staging for Future Phases)

- `Config.General.api_token` (Phase 48) stages the config field Phase 50 will enforce
- `_emit_startup_warnings()` (Phase 48) provides advisory-level warnings Phase 50 will pair with enforcement
