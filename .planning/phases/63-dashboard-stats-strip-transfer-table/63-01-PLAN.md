Create `src/angular/src/app/services/files/dashboard-stats.service.ts`:

1. Define and export an interface `DashboardStats`:
   ```typescript
   export interface DashboardStats {
     downloadingCount: number;
     queuedCount: number;
     totalSpeedBps: number;
     peakSpeedBps: number;
     remoteTrackedBytes: number;
     localTrackedBytes: number;
     totalTrackedBytes: number;
   }
   ```

2. Create `DashboardStatsService` as `@Injectable()` class implementing `OnDestroy`:
   - Inject `ViewFileService`
   - Private `destroy$ = new Subject<void>()`
   - Private `_peakSpeed = 0`
   - Private `_stats$ = new BehaviorSubject<DashboardStats>` initialized with all zeros
   - In constructor, subscribe to `this.viewFileService.files.pipe(takeUntil(this.destroy$))`:
     - Filter files where `status === ViewFile.Status.DOWNLOADING` to get `downloading` list
     - Filter files where `status === ViewFile.Status.QUEUED` to get `queued` list
     - Compute `totalSpeed` as `downloading.reduce((sum, f) => sum + (f.downloadingSpeed ?? 0), 0)`
     - Update `this._peakSpeed = Math.max(this._peakSpeed, totalSpeed)`
     - Compute `remoteTrackedBytes` as `files.reduce((sum, f) => sum + (f.remoteSize ?? 0), 0)`
     - Compute `localTrackedBytes` as `files.reduce((sum, f) => sum + (f.localSize ?? 0), 0)`
     - `totalTrackedBytes = remoteTrackedBytes + localTrackedBytes` (used as denominator for progress)
     - Emit via `this._stats$.next({...})`
   - Public getter `get stats$(): Observable<DashboardStats>` returning `this._stats$.asObservable()`
   - `ngOnDestroy()` calls `this.destroy$.next()` and `this.destroy$.complete()`

3. Register in `src/angular/src/app/app.config.ts`:
   - Add import: `import {DashboardStatsService} from "./services/files/dashboard-stats.service";`
   - Add `DashboardStatsService` to the providers array (after `ViewFileService`)
   - Add APP_INITIALIZER entry using `dummyFactory` pattern (same as ViewFileFilterService) so the service is eagerly created:
     ```typescript
     {
         provide: APP_INITIALIZER,
         useFactory: dummyFactory,
         deps: [DashboardStatsService],
         multi: true
     },
     ```
