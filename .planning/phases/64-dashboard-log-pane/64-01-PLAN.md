## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-64-01 | Information Disclosure | log-pane template | accept | Log pane displays data already visible on the existing Logs page; no new data exposure. Read-only display component. |
| T-64-02 | Tampering | Clipboard writeText | accept | Copy button writes component-local array content to clipboard; no server mutation. try/catch guards non-secure contexts. |
| T-64-03 | Denial of Service | ReplaySubject replay flood | mitigate | Cap entries array at 50 with shift() discard; ReplaySubject replays up to 5000 entries but only last 50 are retained. Array operations are sub-millisecond. |
| T-64-04 | Spoofing | Log message content | accept | Messages are rendered as text content (Angular template interpolation auto-escapes HTML). No innerHTML or bypassSecurityTrust usage. XSS not possible. |
</threat_model>
