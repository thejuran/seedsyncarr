**PYFIX-13 -- test_dispatch.py (per D-03):**

This file has 6 `time.sleep` calls. Two are shutdown-race sleeps (0.5s each = 1.0s savings target), four are stabilization sleeps (0.1s each).

**Shutdown-race sleeps (lines ~229 and ~566):**
These are inside `_extract_archive(**kwargs)` side-effect callbacks that simulate slow extraction work. The test calls `dispatch.stop()` while the extraction is "in progress". Replace `time.sleep(0.5)` with an interruptible `threading.Event.wait(0.5)`.

For each test method that has this pattern (`test_extract_exits_command_early_on_shutdown` and the similar test near line 563):

1. Create a `threading.Event` at the start of the test body (or in the nested scope where `_extract_archive` is defined): `shutdown_event = threading.Event()`
2. In the `_extract_archive` callback, replace `time.sleep(0.5)` with `shutdown_event.wait(0.5)`
3. After the test calls `dispatch.stop()` (or the equivalent shutdown trigger), add `shutdown_event.set()` to release the blocked wait immediately

Example transformation:
```python
# Before:
def _extract_archive(**kwargs):
    print(kwargs)
    self.call_stop = True
    time.sleep(0.5)

# After:
shutdown_event = threading.Event()

def _extract_archive(**kwargs):
    print(kwargs)
    self.call_stop = True
    shutdown_event.wait(0.5)

# ... later, after dispatch.stop() is triggered by the call_stop flag:
shutdown_event.set()
```

Note: Read the actual test flow carefully. In some tests, `self.call_stop = True` signals the main thread to call `dispatch.stop()`. The main thread must call `shutdown_event.set()` after `dispatch.stop()` so the extraction callback's `wait(0.5)` returns immediately. Look for the pattern where the main thread polls `self.call_stop` and then calls stop -- add `shutdown_event.set()` there.

**Stabilization sleeps (lines ~718, 721, 726, 856):**
These `time.sleep(0.1)` calls wait for worker threads to process. Reduce to `time.sleep(0.01)`. This saves 0.36s (4 x 0.09s). Do NOT mock these -- they are timing windows for thread scheduling.
