    Implements PLSH-05 per CONTEXT.md D-10 and D-11.

    D-10 spec: "Target height ~64 px (Material header); resize preserving aspect
    ratio." The source `doc/brand/wordmark-dark.png` is 1376×768. To honor the
    "~64 px" target while also giving high-DPI (Retina) displays crisp rendering,
    downscale to 128 px on the long edge — that yields ~128×71, where 71 is
    within the "~64" tolerance and provides 2× pixel density for the Material
    header's ~64 px display height.

    `sips -Z` caps the longer dimension and scales the other proportionally.
    For a 1376×768 landscape source, `-Z 128` caps width at 128 and produces
    a height of round(768 × 128 / 1376) ≈ 71 px.

    ```
    sips -Z 128 -s format png doc/brand/wordmark-dark.png --out src/python/docs/images/logo.png
    ```
