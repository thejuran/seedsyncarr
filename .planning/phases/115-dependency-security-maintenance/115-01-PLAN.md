## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Dependabot PR branch -> main | Third-party-authored dependency bumps cross into the trunk on merge. Trust is delegated to GitHub CI (the locked safety contract) plus the prior Dependabot ecosystem checks. |
| npm/pip registry -> lockfile | The actual remediation content (undici 7.28.0, hono 4.12.26, piscina, Python dev-deps) originates from public registries; integrity is enforced by the committed lockfile hashes that CI builds against. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-115-01 | Tampering | Merging a PR whose CI is red/stale (regressed build or lint) | mitigate | Hard per-merge precondition (Tasks 2-3): `gh pr checks <N>` every blocking gate pass/SUCCESS AND `gh pr view` mergeable=MERGEABLE/CLEAN immediately before each `gh pr merge --squash`; never `--admin`/force past a red gate. |
| T-115-02 | Tampering | Out-of-order / batched merges hiding a bad interaction | mitigate | One-at-a-time squash merges in locked order #64->#65->#66->#60->#61->#62->#63 (D-01/D-03/D-04); no `gh pr merge --auto`; bounded wait for Dependabot rebase before merging a behind PR (D-02). |
| T-115-03 | Tampering | Supply-chain: a Dependabot PR carries a malicious/typo'd transitive update | accept | These are first-party Dependabot bumps of already-present deps (no new packages introduced); integrity enforced by committed lockfile hashes + full CI build/test/E2E gate per PR. No `## Package Legitimacy Audit` needed — phase introduces no net-new package, only version bumps of existing deps. Residual risk low for a single-user self-hosted tool. |
| T-115-04 | Information Disclosure | New ruff 0.15.17 rule masks/auto-rewrites code in an unsafe way | mitigate | Local whole-tree `ruff check` only (no `--fix`/`--unsafe-fixes`); new findings surfaced to operator, never auto-suppressed (D-07, Task 4). |
| T-115-SC | Tampering | npm/pip installs (`pip install 'ruff==0.15.17'`, Dependabot npm group) | mitigate | No net-new packages — only version bumps of existing deps already vetted in prior Dependabot cleanups; lockfile-hash integrity + full CI gate per merge.
